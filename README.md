# Music Player App - Solid.js

A modern music player application built with Solid.js.

## Features

- Music playback controls
- Playlist management
- Progress tracking
- Song selection
- Modern, responsive UI

## Installation

```bash
npm install
```

## Running

```bash
# Development
npm run dev

# Production
npm run build
npm run preview
```

## Tech Stack

- Solid.js
- TypeScript
- Vite
